To create a new conversation:

1. Navigate to Assets/ScriptableObjects/UI/Dialogue
2. In the Dialogue folder, right click in the project window > Create
   > Dialogue > NewConversation
3. Name the newly created conversation
4. Select the conversation and fill out the character exchanges
5. Attach the DialogueTrigger script to the object that will trigger
   the dialogue and attach the conversation object to it in the inspector
